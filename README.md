# @hawiah/local [![NPM version](https://img.shields.io/npm/v/@hawiah/local.svg?style=flat-square&color=informational)](https://npmjs.com/package/@hawiah/local)

Local file storage drivers for JSON and YAML with atomic writes using writerx.

## Installation

```bash
npm install @hawiah/local
```

## Features

- Lightweight and fast local storage
- Atomic writes with writerx
- Support for JSON and YAML formats
- Unified IDriver interface
- Auto-save or manual persistence
